一个用 Python 编写的命令行石头剪刀布游戏，支持人机对战和胜率统计。

## ✨ 功能

- ✊✌️✋ 石头/剪刀/布 三种出拳
- 🤖 电脑随机出拳
- 📊 实时统计胜负记录和胜率
- 🔄 异常处理非法输入

## 🚀 快速开始

### 依赖
```bash
无额外依赖（仅用标准库）
```

### 运行
```bash
python rock_scissors_paper.py
```

## 📖 使用说明

运行后输入 1/2/3 出拳，输入 `q` 退出并查看最终战绩。

## 🛠️ 用到的知识点

| 知识点 | 应用 |
|--------|------|
| random模块 | 电脑随机出拳 |
| 列表(list) | 存储拳法选项 |
| 函数定义 | 拆分胜负判断逻辑 |
| if/elif | 映射输入到拳法 |
| 计数器变量 | 统计胜负平场次 |
