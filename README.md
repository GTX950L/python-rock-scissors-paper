一个用 Python 编写的命令行石头剪刀布游戏，支持人机对战和胜率统计。

## 🎮 玩法

- 输入 1/2/3 出拳（石头/剪刀/布）
- 电脑随机出拳
- 实时统计胜负记录和胜率
- 输入 q 退出并查看最终战绩

## ⚡ 快速开始

```bash
python rock_scissors_paper.py
```

## 🧠 学到的知识点

| 知识点 | 应用 |
|--------|------|
| random模块 | 电脑随机出拳 |
| 列表(list) | 存储拳法选项 |
| 函数定义 | 拆分胜负判断逻辑 |
| if/elif | 映射输入到拳法 |
| 计数器变量 | 统计胜负平场次 |
| 百分数计算 | 显示胜率 |
